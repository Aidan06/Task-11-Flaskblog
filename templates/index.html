Добро пожаловать в API Backend

Добро пожаловать на начальную страницу вашего API Backend web-приложения.

Вы можете использовать следующие эндпоинты для взаимодействия с данными:

  • GET /posts - Получить все посты
  • POST /posts - Создать новый пост
  • GET /posts/{post_id} - Получить информацию о конкретном посте
  • PUT /posts/{post_id} - Обновить информацию о конкретном посте
  • DELETE /posts/{post_id} - Удалить конкретный пост
  • POST /posts/{post_id}/comments - Создать новый комментарий для поста
  • GET /posts/{post_id}/comments - Получить все комментарии для поста
  • DELETE /comments/{comment_id} - Удалить конкретный комментарий
